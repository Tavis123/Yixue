# Yixue
一个基于springcloud的微服务自学网站

主要业务如下：
- 用户端
    - 登录
    - 注册
    - 课程搜索
    - 课程推荐
    - 视频学习
    - 课程评论
    - 课程问答
    - 学习作业
    - 我的考试
    - 我的订单
    - 个人中心
- 机构端
    - 入驻加盟
    - 内容管理
    - 教师管理
    - 作业管理
    - 考试管理
    - 评论管理
    - 订单管理
    - 媒资管理
    - 统计分析
    - 活动管理
- 平台端
    - 机构管理
    - 分类管理
    - 课程审核
    - 媒资审核
    - 题目审核
    - 用户管理
    - 统计分析
    - 权限配置
    - 财务管理
    - 系统管理
